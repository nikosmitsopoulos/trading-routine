# Watchlist

Stocks under consideration. Each entry must have a full thesis before it can move to a real trade.

Format:
```
## TICKER — Added YYYY-MM-DD

- **Company:** Full name, sector
- **Thesis:** 2-3 sentences on why this could work.
- **Catalysts:** What near-term events could move the stock?
- **Buy signals present:** (which from strategy.md)
- **Valuation:** P/E, PEG, etc.
- **Entry price target:** $X.XX (or "current")
- **Conviction:** low/medium/high
- **Stop if bought:** $X.XX
- **Status:** watching / ready / passed
```

Prune entries after 14 days if not acted on.
